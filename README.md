# Geometry Calculator (C++)

A C++ console application that calculates the areas and volumes of common geometric shapes with input validation and formatted output.

## Features

### Area Calculations

* Triangle
* Rectangle
* Circle
* Square

### Volume Calculations

* Cuboid
* Cylinder
* Sphere

## Input Validation

The program ensures:

* Only numeric input is accepted
* Negative values are rejected
* Zero values are rejected
* Users are prompted until valid input is entered

## Concepts Used

* Functions
* Constants (`constexpr`)
* Input validation
* Loops
* Mathematical formulas
* Formatted output using `iomanip`
* Modular programming
